# Hellenic Open Data — Ψηφιακό Ευρετήριο

Μικρό ευρετήριο πόρων για την αρχαία ελληνική γλώσσα: λεξικά, κείμενα και εργαλεία γραμματικής.

Περιεχόμενα
- `index.html` — κύρια σελίδα
- `style.css` — στυλ
- `script.js` — λογική της σελίδας (φορτώνει δεδομένα από `data.json`)
- `data.json` — δεδομένα των πόρων

Τοπική εκτέλεση
1. Άνοιξε το `index.html` σε browser (διπλό κλικ ή `Live Server`).

Συμβολή
- Φτιάξε ένα νέο branch, κάνε αλλαγές και άνοιξε pull request.
